# 📦 MVP Scope & Feature Bucketing (LU 9)

## 1. 🟢 MVP Bucket (Must-Have)
*These features solve the core fragmentation problem.*

- **Auth (JWT)** — Necessary to keep the community exclusive and professional (Kalvium email only).
- **Announcements Feed** — Solves the "Muted WhatsApp group" problem for campus managers.
- **Notes & Resources Module** — Directly addresses the loss of high-value semester materials.
- **Placement Experiences** — Captures unique 3-4 round interview patterns before they are forgotten.
- **Search & Filter (Semester/Uni)** — The core "Discoverability" layer that differentiates this from a chat app.
- **Basic Discussion Forum** — Provides a space for peer-to-peer technical help.

---

## 2. 🟡 Post-MVP Bucket (Nice-to-Have)
*These can proceed after the core habit is established.*

- **AI-Note Summarization** — While high-value, it's a multiplier, not a baseline requirement.
- **Hackathon Application Logic** — The MVP can initial show "Hackathon Posts," but complex application workflows are complex to build early.
- **Upvote/Reward Badges** — Nice for "gamification" but can follow once there's a critical mass of users.
- **Mentor Bookings** — 1-on-1 scheduling is a heavy feature that distracts from baseline "Sharing."

---

## 3. 🔴 Cut Bucket (Explicitly Out of Scope)
*These will NOT be built to maintain ruthless focus.*

- **Real-time DM/Chat** — **Reason:** WhatsApp already does this; building a chat engine in an MVP is a distractions from building a "Knowledge System."
- **Native Android/iOS App** — **Reason:** Kalvium students primarily work on laptops; a PWA (Progressive Web App) is faster and cheaper for the first 1,000 users.
- **Video Hosting/Streaming** — **Reason:** Storing 1GB+ lecture videos is extremely expensive; the product will only support external embedding (YouTube/Drive).
- **Public SEO Access** — **Reason:** This is a Kalvium-exclusive private platform; keeping it closed builds trust for sharing sensitive placement tips.

---

## 💎 Core Value Summary
The **Kalvi Connect MVP** delivers a structured, searchable, and permanent "Living Library" for Kalvium students, replacing the messy, transient nature of WhatsApp threads. It focuses on **Academic Success (Notes)** and **Career Outcomes (Placement Experiences)** while deliberately ignoring social chat features and complex administrative micro-management to ensure we ship a high-quality "Knowledge Hub" first.
